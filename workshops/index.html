Workshops

Each day of the Bootcamp, there will be four workshop sessions--two in the morning and two in the afternoon, with breaks between each session. Workshop topics are scaffolded so that successive workshops will use skills developed in earlier workshops. All attendees will participate in the same core workshops, and then will be able to choose between different tracks for the advanced workshops.

Workshop Details

Command Line

Introduction to the UNIX command line. Topics covered will include the filesystem, environment, useful commands, working with files and folders, input and output using pipes and simple command line programs.

Git

Git is a tool for managing changes to a set of files. It allows users to store new changes, recover old versions reliably, and helps manage conflicting changes made by different users.

Python

Python is a programming language that can be used for a wide range of tasks, from collecting data to analysis to building web applications.

Databases

Store complex data using databases and manipulating, analyzing, and searching databases using SQL.

Scientific Python

Manage and analyse tabular data using Pandas, and work with N-dimensional array data using numpy and scipy.

HTML/CSS/Javascript

Implement a modern website that uses HTML for content, CSS to control appearance, and JavaScript to dictate behavior.

Data Cleaning

Dealing with some common problems with datasets, including file types and conversions, inconsistent metadata, normalization, and working with APIs.

Data Visualization

Data visualization using either programmatic or interactive tools, including basic geospatial visualization

NLTK

NLTK supports classification, tokenization, stemming, tagging, parsing, and semantic reasoning, and has easy-to-use interfaces to corpora and lexical resources.

Digital Identity/Wordpress

Build a wordpress site to control online branding and learn about various add-ons to make the site work for you.

GIS

Discussion of geographic data, the various ways of representing it, and methods for analysing and visualizing spatial data.

Machine Learning

Regression analysis, data preprocessing, model selection, supervised and unsupervised machine learning and dimension reduction using the sklearn machine learning library.
